dotProject Gallery2 Integrationd 
--------------------------------
Integrates a gallery2 site in your dotproject installation.

Vegard Fiksdal -- fiksdal@sensorlink.no
2007(c)Sensorlink AS

Prerequisites
-------------
First you need to install and configure a gallery2 site. This site must be
accessible from the network, and verified to work before you proceed. Check
the gallery2 webpage for downloads and information: 

http://gallery.menalto.com/

Installation
------------
Install the module as you would any other module, and click 'configure' to
link dotproject with your gallery site. Dotproject need to know the URI for
your gallery2 site, the local path of it, and optionally a user to log in
whith (Empty=Anonymous).
